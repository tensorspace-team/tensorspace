下载
TensorSpace.js可以很好地运行在 Chrome, safari, firefox 浏览器中。在我们的官网中包含了 TensorSpace.js 文档，文件下载和使用样例。

TensorSpace.js 的源码可以从 TensorSpace 的 Github 仓库下载。如果有关于 TensorSpace 的使用问题，可以到 github 上的 issue 部分提出宝贵意见。

使用NPM下载


    $ npm install tensorspace

使用Yarn下载


    $ yarn add tensorspace

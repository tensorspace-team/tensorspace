Multiply
"Multiply" is constructed as a "factory" to complete the "multiply" operation. To apply the "Multiply" function, it is required that the shapes of all input layers are the same (otherwise, we can't perform the operation). The "factory" returns a merged layer instance as the output.
Factory style
layerMultiplyResult = TSP.layers.Multiply( [layer1, layer2, ...], { configs } )
The first parameter is a list of layer objects. (TensorSpace currently only supports "Multiply" operations for 3D vectors. We will introduce others soon.)
The second parameter is the configurations of the output merged layer object.

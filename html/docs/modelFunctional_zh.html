函数式模型
Model() 能用来创建 TensorSpace 模型。 与 Sequential() 创建简单的层对象堆栈不同， Model() 需要提供关于每一层关系的细节信息。
我们可以按照以下方式创建一个函数模型：
let model = new TSP.model.Model( container, {

    ...
    // necessary configurations
    inputs: [input],
    outputs: [addLayer],
    outputsOrder: ["input", "conv1", "conv2", "conv3", "add"],
    layerShape: "rect",
    layerInitStatus: "close",
    stats: true
    ...

} );
我们可以通过添加层与层之间的对应输入输出关系来逐步地向模型中添加层对象。 
let input = new TSP.layers.GreyscaleInput({
        shape: [28, 28, 1]
});

let layer1 = new TSP.layers.Conv2d({
	kernelSize: 2,
	filters: 36,
	strides: 1,
	padding: "same"
});

layer1.apply( input );

let layer2 = new TSP.layers.Conv2d({
	kernelSize: 2,
	filters: 36,
	strides: 1,
	padding: "same"
});

layer2.apply( input );

let layer3 = new TSP.layers.Conv2d({
	kernelSize: 2,
	filters: 4,
	strides: 1,
	padding: "same"
});

layer3.apply( layer2 );

let addLayer = new TSP.layers.Add([layer1, layer3]);

let model = new TSP.model.Model(container, {

    ...
    inputs: [input],
    outputs: [addLayer],
    outputsOrder: ["input", "conv1", "conv2", "conv3", "add"],
    ...

});
在完成创建模型结构之后，我们可以使用 load() 载入所对应的神经网络模型，并通过 init() 初始化： 
model.load( {

    type: "tfjs",
    url: './lenetModel/mnist.json',

    onComplete: function() {

        console.log( "Complete load model." );

    }

} );

model.init();
predict() 方法方法将接管所有层对象关于数据预测的处理。所有结果将被用于渲染层对象。
Property
.inputs : Layer[]
  • filter_center_focus 必要. 只可用于Model()。
  • filter_center_focus 表示该模型的输入层对象。
  • filter_center_focus 如果模型使用多输入，我们必须使用 Model() 来创建模型。
.outputs : Layer[]
  • filter_center_focus 必要. 只可用于Model()。
  • filter_center_focus 表示该模型的输出层对象。
  • filter_center_focus 模型的输出可以为多个。
.outputsOrder : String[]
  • filter_center_focus 必要. 只可用于Model()。
  • filter_center_focus 表示输出层对象的名称列表。
  • filter_center_focus 展示输出层对象的输出顺序。
  • filter_center_focus 其输出顺序代表了预处理的神经网络模型的中间层计算输出的顺序。
.aggregationStrategy : String
  • filter_center_focus 表示聚合体表面渲染的策略。
.layerShape : String
  • filter_center_focus 表示特征图的排布方式。
.layerInitStatus : String
  • filter_center_focus 表示层初始化状态：展开为特征图或者收拢为聚合体。
.textSystem : String
  • filter_center_focus 启用或者禁用层的文字显示功能。
.relationSystem : String
  • filter_center_focus 启用或者禁用层间关系连线的显示功能。
.animationTimeRatio : Double
  • filter_center_focus 实际动画时间与默认动画时间的比值。
.color : color format
  • filter_center_focus 关于所有层对象的颜色配置以及场景背景颜色的配置。
.stats : Boolean
  • filter_center_focus 是否启用并显示 stats 控件。
Method
.init( callbackFunction ) : void
  • filter_center_focus init( callbackFunction )方法使用所提供的配置信息创建模型中所有对象。
  • filter_center_focus 详情参见Model (concept)。
.load( model_config ) : void
  • filter_center_focus load() 方法用于载入神经网络模型。
  • filter_center_focus 详情参见Load 。
.predict( input_data ) : void
  • filter_center_focus predict() 方法将收集所有必要的层间数据并将其渲染至所对应的可视化对象中。
  • filter_center_focus 详情参见Predict。
.getAllLayers() : Layer[]
  • filter_center_focus 以获取所有层对象。
.getLayerByName( name ) : Layer
  • filter_center_focus 根据所提供的层名称（String）来获取特定的层对象。

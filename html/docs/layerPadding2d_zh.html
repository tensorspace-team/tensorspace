Padding2d

对前一层进行padding操作，在矩阵（二维向量）的四周补零

构造器 3️⃣

TSP.layers.Padding2d( { padding : [ Int, Int ] } )

参数列表

参数名
标签		 类型 简介 具体用法细节和例子
padding
⭐️⚙️		 [Int] 四周加0的数量 例如，padding=[ 2, 2 ]，第一个值表示向左右分别加2个零，第二个值表示向上下加2个0
name
🔧		 String 层的命名，建议添加 name: "layerName"
color
🎨⚙️		 color format 层的颜色 Padding2d默认颜色是浅蓝色
closeButton*
🎨⚙️		 Dict 层关闭按钮外观控制列表 display : Bool true [default] 显示按钮 false 隐藏按钮
ratio : Int 为正常大小的几倍，默认为1倍
例如，设为2为正常大小的2倍大
initStatus
🎦⚙️		 String 初始化时，本层是否收缩 close[default] : 收缩
animationTimeRatio
🎦⚙️		 Int 张开和伸缩的调整速度，呈倍速关系 例如2就是2倍，数字越大速度越快

*closeButton 例子

closeButton {
    display: false,
    ratio: 2,
}

标签详情

符号 参数性质 说明
⭐️ 必要 使用时必须提供，不能为空
只要提供了这些参数，就能正常创建实例，控制参数使用默认值
🔧 推荐 使用时推荐给定，未给定也可以运行，但在体验和易用性上有隐患
⚙️ 可选 根据模型配置参数选择性添加
这里的参数对于层的结构（3D可视化外观）没有影响
📦 模型 配置层的相关参数，和模型设计有关
🎨 外观 可覆盖TSP.model.Sequential下的属性进行细调，详情点击
🎦 动画控制 可覆盖TSP.model.Sequential下的属性进行细调，详情点击

调用结果

  • 创建一个新的对象，往3D场景中添加一个padding层
  • 默认颜色: #6eb6ff

属性

.inputShape : [Int]

在 model.init() 后才可拿到数据，否则为undefined

本层输入Tensor的形状。Dataformat默认通道值在最后，例如inputShape = [ 28, 28, 3 ] 表示输入为3个特征图，每个图大小为28*28

.outputShape : [Int]

在 model.init() 后才可拿到数据，否则为undefined

本层输出Tensor的形状为3维，例如outputShape = [ 32, 32 ] 表示经过此层处理后，输出尺寸是32*32

.neuralValue : [float32]

载入模型，在 model.predict() 后才可以拿到输入数据，否则为undefined

本层的层间输出值数组，padding层不改变输入的值，只是在边缘加0，输出的形状改变

.name : String

创建后即可取到

本层的自定义名称

.layerType : String

创建后即可取到

本层的类型，返回一个定值，字符串Padding2d

方法

.openLayer() : void

（1）通过直接和3d场景中物体交互直接点击层打开

（2）代码中通过调用方法打开

model.init();
TSP.layers.Padding2d.openLayer()

.closeLayer() : void

（1）通过直接和3d场景中物体交互点击按钮关闭

（2）代码中通过调用方法打开

model.init();
// if this layer already opened
TSP.layers.Padding2d.closeLayer()

使用样例

添加层

（1）声明一个Padding2d的实例，在左右加2个0，上下加3个0，方便复用

let paddingLayer = new TSP.layers.Padding2d({
    padding: [ 2, 3 ],
    name: "Padding2d1",
    initStatus: "open",
});
model.add(paddingLayer);

（2）直接添加Padding2d

model.add(new TSP.layers.Padding2d({
    padding: [ 2, 3 ],
    name: "Padding2d2"
}));
